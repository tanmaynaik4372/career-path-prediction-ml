# Career Path Prediction using Machine Learning

## 📌 Project Overview
This MSc dissertation project develops a machine learning system to predict suitable career paths based on individual skills, academic background, and preferences.

## 📊 Techniques Used
- Data Cleaning & Preprocessing
- Feature Engineering
- Classification Models (Scikit-Learn)
- Model Evaluation (Accuracy, Precision, Recall, F1-score)

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-Learn
- Jupyter Notebook

## 📈 Outcome
The model was evaluated using multiple performance metrics to support data-driven career guidance decisions.
